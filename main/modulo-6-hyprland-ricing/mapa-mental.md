## рҹ”‘ Conceptos Clave

```

  в”Ӯ   в”ңв”Җв”Җ X11: protocolo legacy, display server
  в”Ӯ   в”ңв”Җв”Җ Wayland: protocolo moderno, mГЎs seguro

  в”ңв”Җв”Җ Compositor (Window Manager)
  в”Ӯ   в”ңв”Җв”Җ Gestiona ventanas y su disposiciГіn
  в”Ӯ   в”ңв”Җв”Җ Tiling: ventanas organizadas automГЎticamente
  в”Ӯ   в””в”Җв”Җ Animaciones y efectos visuales
  в”ңв”Җв”Җ Plugins C++
  в”Ӯ   в”ңв”Җв”Җ Hyprland soporta plugins nativos
  в”Ӯ   в”ңв”Җв”Җ API del compositor
  в”Ӯ   в””в”Җв”Җ Compilar contra las cabeceras de Hyprland
  в”ңв”Җв”Җ IPC (Inter-Process Communication)
  в”Ӯ   в”ңв”Җв”Җ hyprctl вҖ” controlar Hyprland vГӯa CLI
  в”Ӯ   в”ңв”Җв”Җ Sockets UNIX
  в”Ӯ   в””в”Җв”Җ Automatizar acciones del escritorio
  в”ңв”Җв”Җ QML (Qt Modeling Language)
  в”Ӯ   в”ңв”Җв”Җ Declarativo para UIs
  в”Ӯ   в”ңв”Җв”Җ Widgets personalizados
  в”Ӯ   в””в”Җв”Җ IntegraciГіn con datos del sistema
  в”ңв”Җв”Җ 9-slice Scaling
  в”Ӯ   в”ңв”Җв”Җ TГ©cnica para escalar bordes de ventanas
  в”Ӯ   в”ңв”Җв”Җ Esquinas fijas, bordes estirables
  в”Ӯ   в””в”Җв”Җ ComГәn en temas y decoraciones
  в””в”Җв”Җ Rope Physics (simulaciГіn de cuerda)
```
